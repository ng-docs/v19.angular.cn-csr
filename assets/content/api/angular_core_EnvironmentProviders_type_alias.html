描述

封装的 Provider，仅在创建 EnvironmentInjector 时（例如在 NgModule 中）被接受。

使用此包装器类型可以防止那些仅设计用于应用/环境注入器的提供者被意外地包含在 @Component.providers 中，并最终进入组件注入器。

此包装器类型可防止访问内部的 Provider。
