@angular/core

APP_ID

常量
代码

一个 DI 令牌，表示字符串 ID，主要用于在使用 ViewEncapsulation#Emulated 时为应用属性和 CSS 样式添加前缀。

API

  
    const APP_ID: InjectionToken<string>;

说明

一个 DI 令牌，表示字符串 ID，主要用于在使用 ViewEncapsulation#Emulated 时为应用属性和 CSS 样式添加前缀。

在同一页面上引导启动多个应用（例如，使用 bootstrapApplication 调用）时，需要此令牌。在这种情况下，请确保这些应用具有不同的 APP_ID 值设置。例如：

          
bootstrapApplication(ComponentA, {  providers: [    { provide: APP_ID, useValue: 'app-a' },    // ... other providers ...  ]});bootstrapApplication(ComponentB, {  providers: [    { provide: APP_ID, useValue: 'app-b' },    // ... other providers ...  ]});

默认情况下，当只有一个应用被引导启动时，你不需要提供 APP_ID 令牌（ng 将用作应用 ID）。
