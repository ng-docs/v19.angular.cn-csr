Angular 平台是 Angular 在网页上的入口点。 每个页面只有一个平台。诸如反射之类的服务，对于在该页面上运行的每个 Angular 应用都是通用的，并在其作用域内绑定。 当使用平台工厂（例如 PlatformBrowser）创建平台时，页面的平台会被隐式初始化；或者通过显式调用 createPlatform() 函数来初始化。
