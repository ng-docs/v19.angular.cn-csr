ResolveData

类型别名
代码

表示与特定路由关联的已求解数据。

Route#resolve

API

  
    type ResolveData = {  [key: string | symbol]: ResolveFn<unknown> | DeprecatedGuard;}

说明

表示与特定路由关联的已求解数据。

返回 RedirectCommand 会指示路由器取消当前导航并重定向到 RedirectCommand 中提供的位置。请注意，当解析器执行时，没有排序保证。如果多个解析器都返回 RedirectCommand，则只会使用返回的第一个。
