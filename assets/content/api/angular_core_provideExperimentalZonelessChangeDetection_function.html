provideExperimentalZonelessChangeDetection

EnvironmentProviders

为使用 bootstrapApplication 引导的应用提供不带 ZoneJS 的变更检测。

此函数允许你配置应用，使其不使用 ZoneJS 的状态/状态更改来调度应用中的变更检测。当页面上根本没有 ZoneJS，或者由于其他原因（另一个使用 ZoneJS 进行调度的 Angular 应用或某些依赖 ZoneJS 的其他库）而存在 ZoneJS 时，此函数将起作用。

也可以将其添加到 TestBed 提供者中，以配置测试环境，使其更接近生产行为。这将有助于提高组件与不带 Zone 的变更检测兼容的信心。

ZoneJS 使用浏览器事件来触发变更检测。当使用此提供者时，Angular 将改为使用 Angular API 来调度变更检测。这些 API 包括：

  • ChangeDetectorRef.markForCheck
  • ComponentRef.setInput
  • 更新在模板中读取的信号
  • 当绑定的宿主或模板监听器被触发时
  • 附着一个被上述操作之一标记为脏的视图
  • 移除视图
  • 注册渲染钩子（仅当渲染钩子执行上述操作之一时，模板才会刷新）
