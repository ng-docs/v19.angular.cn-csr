ControlValueAccessor 用于写入数字值和监听数字输入变更。 值访问器由 FormControlDirective、FormControlName 和 NgModel 指令使用。

描述

ControlValueAccessor 用于写入数字值和监听数字输入变更。 值访问器由 FormControlDirective、FormControlName 和 NgModel 指令使用。

导出自

  • ReactiveFormsModule
  • FormsModule

用法说明

将数字输入与响应式表单一起使用。

以下范例展示了如何将数字输入与响应式表单一起使用。

          
const totalCountControl = new FormControl();

        
          
<input type="number" [formControl]="totalCountControl">
