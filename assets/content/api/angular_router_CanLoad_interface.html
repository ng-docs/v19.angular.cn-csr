描述

类可以实现的接口，以用作守卫，决定是否可以加载子路由。 如果所有守卫都返回 true，则导航继续。如果任何守卫返回 false，则导航取消。如果任何守卫返回 UrlTree，则当前导航取消，并开始新的导航到守卫返回的 UrlTree。

以下范例实现了一个 CanLoad 函数，该函数用于决定当前用户是否有权加载所请求的子路由。
