RxResourceOptions

interface
实验性的
代码

类似于 ResourceOptions，但使用基于 RxJS 的 loader。

API

  
    interface RxResourceOptions<T, R> extends BaseResourceOptions<T, R> {}

loader

(params: ResourceLoaderParams<R>) => Observable<T>

request

(() => R) | undefined

一个响应式函数，用于确定要发出的请求。每当请求更改时，将触发 loader 以获取资源的新值。

如果未提供请求函数，则除非重新加载资源，否则 loader 不会重新运行。

defaultValue

NoInfer<T> | undefined

当服务器值不可用时（例如，当资源仍在加载或处于错误状态时），将从资源返回的值。

equal

ValueEqualityFn<T> | undefined

用于比较 loader 的返回值的相等性函数。

injector

Injector | undefined

重写 resource 使用的 Injector。
