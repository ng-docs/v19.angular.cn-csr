描述

ControlValueAccessor 用于写入多选控件值和监听多选控件的更改。值访问器被 FormControlDirective、FormControlName 和 NgModel 指令使用。

用法说明

使用多选控件

以下范例展示了如何将多选控件与响应式表单一起使用。

          
const countryControl = new FormControl();

        
          
<select multiple name="countries" [formControl]="countryControl">  <option *ngFor="let country of countries" [ngValue]="country">    {{ country.name }}  </option></select>

自定义选项选择

要自定义默认选项比较算法，<select> 支持 compareWith 输入。 请参阅 SelectControlValueAccessor 以了解用法。
