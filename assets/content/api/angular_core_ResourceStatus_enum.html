ResourceStatus

枚举
实验性的
代码

Resource 的状态。

API

  
    enum ResourceStatus {}

空闲

资源没有有效的请求，将不会执行任何加载。

value() 将为 undefined。

错误

加载失败并出现错误。

value() 将为 undefined。

加载中

由于其 request 发生更改，资源当前正在加载新值。

value() 将为 undefined。

重新加载中

资源当前正在为相同的请求重新加载一个全新的值。

value() 将在重新加载操作期间继续返回先前获取的值。

已求解

加载已完成，并且资源具有从加载器返回的值。

本地

资源的值是通过 .set() 或 .update() 在本地设置的。
