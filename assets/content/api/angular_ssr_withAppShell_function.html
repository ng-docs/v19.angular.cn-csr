withAppShell

ServerRoutesFeature<ServerRoutesFeatureKind>

使用提供的组件配置 app shell 路由。

App shell 充当应用的主要入口点，通常用于启用应用外壳的服务端渲染 (SSR)。它处理与任何特定服务器路由都不匹配的请求，提供了一种回退机制，并提高了导航期间的感知性能。

这种配置在利用渐进式 Web 应用 (PWA) 模式（例如 Service Worker）的应用中尤其有用，以提供无缝的用户体验。
