@angular/platform-browser

createApplication

函数
代码

创建一个 Angular 应用的实例，而无需引导任何组件。这在希望将应用环境的创建（平台和相关的注入器）与屏幕上渲染组件分离开来的情况下非常有用。组件随后可以在返回的 ApplicationRef 上进行引导。

API

  
    function createApplication(  options?: ApplicationConfig | undefined): Promise<ApplicationRef>;

createApplication

Promise<ApplicationRef>

创建一个 Angular 应用的实例，而无需引导任何组件。这在希望将应用环境的创建（平台和相关的注入器）与屏幕上渲染组件分离开来的情况下非常有用。组件随后可以在返回的 ApplicationRef 上进行引导。

@paramoptionsApplicationConfig | undefined

应用环境的额外配置。有关更多信息，请参阅 ApplicationConfig。

@returnsPromise<ApplicationRef>
