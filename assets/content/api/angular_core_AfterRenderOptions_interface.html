AfterRenderOptions

interface
开发者预览版
代码

传递给 afterRender 和 afterNextRender 的选项。

API

  
    interface AfterRenderOptions {}

injector

Injector | undefined

在创建期间使用的 Injector。

如果未提供，则将使用当前的注入上下文（通过 inject）。

manualCleanup

boolean | undefined

此 Hook 是否应要求手动清理。

如果这是 false（默认值），则此 Hook 将自动注册自身，以便使用当前的 DestroyRef 进行清理。

phase

AfterRenderPhase | undefined
@deprecated

通过传递一个 spec-object 作为 afterRender 或 afterNextRender 的第一个参数（而不是函数），来指定回调函数应在哪个阶段运行。

回调函数应在其中调用的阶段。

默认为 AfterRenderPhase.MixedReadWrite。你应该选择一个更具体的阶段。有关更多信息，请参阅 AfterRenderPhase。
