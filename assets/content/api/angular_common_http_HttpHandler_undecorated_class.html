描述

将 HttpRequest 转换为 HttpEvent 流，其中一个很可能是 HttpResponse。

HttpHandler 是可注入的。当被注入时，处理器的实例会将请求分发到链中的第一个拦截器，然后分发到第二个拦截器，依此类推，最终到达 HttpBackend。

在 HttpInterceptor 中，HttpHandler 参数是链中的下一个拦截器。
