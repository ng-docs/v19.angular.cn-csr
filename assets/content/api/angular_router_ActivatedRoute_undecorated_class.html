描述

提供对与 outlet 中加载的组件关联的路由信息的访问。 用于遍历 RouterState 树并从节点中提取信息。

以下范例展示了如何使用来自当前激活路由的信息来构建组件。

注意：此class中的可观察对象仅在当前值和先前值基于浅相等性而不同时发出。例如，更改已求解的 data 中的深层嵌套属性不会导致 ActivatedRoute.data Observable 发出新值。
