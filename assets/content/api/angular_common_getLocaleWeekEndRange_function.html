getLocaleWeekEndRange

[WeekDay, WeekDay]

对于给定的语言环境，被认为是周末的星期范围。

@deprecated

Angular 建议依赖 Intl API 进行 i18n。 Intl 的 getWeekInfo 具有部分支持（Chromium M99 和 Safari 17）。 诸如 Luxon 之类的库依赖于 Intl，但如果不支持 getWeekInfo，则回退到 ISO 8601 定义（星期六+星期日）。

@paramlocalestring

用于语言环境格式规则的语言环境代码。

@returns[WeekDay, WeekDay]
