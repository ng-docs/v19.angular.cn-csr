CreateEffectOptions

开发者预览
代码

传递给 effect 函数的选项。

API

  
    interface CreateEffectOptions {}

injector

Injector | undefined

在其中创建副作用的 Injector。

如果未提供，则将使用当前的注入上下文（通过 inject）。

manualCleanup

boolean | undefined

effect 是否应要求手动清理。

如果为 false（默认值），则副作用将自动注册自身，以便使用当前的 DestroyRef 进行清理。

forceRoot

true | undefined

始终创建一个根副作用（计划为微任务），无论 effect 是否在组件内调用。

allowSignalWrites

boolean | undefined
@deprecated

不再需要，默认情况下允许信号写入。

debugName

string | undefined

副作用的调试名称。在 Angular DevTools 中用于标识副作用。
