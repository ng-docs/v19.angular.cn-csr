sequence

AnimationSequenceMetadata

定义要按顺序逐个运行的动画步骤列表。

@paramstepsAnimationMetadata[]

动画步骤对象数组。

  • 由 style() 调用定义的步骤会立即应用样式数据。
  • 由 animate() 调用定义的步骤会在一段时间内应用样式数据，具体时间由时序数据指定。
          
sequence([style({ opacity: 0 }),animate("1s", style({ opacity: 1 }))])
@paramoptionsAnimationOptions | null

一个 options 对象，其中包含延迟和开发者定义的参数，这些参数提供样式默认值，并且可以在调用时被重写。

@returnsAnimationSequenceMetadata

用法说明

当您将步骤数组传递给 transition() 调用时，默认情况下这些步骤会按顺序运行。将其与 group() 调用进行比较，后者并行运行动画步骤。

当序列在 group() 或 transition() 调用中使用时，仅当每个内部动画步骤都完成后，执行才会继续执行下一个指令。
