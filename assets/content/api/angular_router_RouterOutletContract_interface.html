描述

一个接口，为开发 Router 的组件出口定义了契约。

出口充当占位符，Angular 会根据当前的路由器状态动态地填充它。

路由器出口应该通过 ChildrenOutletContexts#onChildOutletCreated 向 Router 注册自己，并通过 ChildrenOutletContexts#onChildOutletDestroyed 注销。当 Router 识别到匹配的 Route 时，它会在 ChildrenOutletContexts 中查找已注册的出口并激活它。
