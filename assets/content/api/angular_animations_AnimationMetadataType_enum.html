@angular/animations

AnimationMetadataType

枚举
代码

可以为动画定义的参数类别常量。

API

  
    enum AnimationMetadataType {}

状态

将命名的动画状态与一组 CSS 样式关联起来。 请参阅 state()

过渡

用于表示从一个动画状态到另一个动画状态的过渡的数据。 请参阅 transition()

序列

包含一组动画步骤。 请参阅 sequence()

组

包含一组动画步骤。 请参阅 <a href="/api/animations/group"><code>group()</code></a>

动画

包含一个动画步骤。 请参阅 animate()

关键帧

包含一组动画步骤。 请参阅 keyframes()

样式

将一组 CSS 属性-值对包含到命名的样式中。 请参阅 style()

触发器

将动画与可以附着到元素上的进入触发器相关联。 请参阅 trigger()

引用

包含一个可复用的动画。 请参阅 animation()

AnimateChild

包含用于执行查询返回的子动画的数据。 请参阅 animateChild()

AnimateRef

包含可复用动画的动画参数。 请参阅 useAnimation()

查询

包含子动画查询数据。 请参阅 query()

交错

包含用于交错动画序列的数据。 请参阅 stagger()

描述

可以为动画定义的参数类别常量。

相应的函数为每个类别定义一组参数，并将它们收集到相应的 AnimationMetadata 对象中。
