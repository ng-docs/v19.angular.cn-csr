验证表单

edit

使用表单时，另一个常见的场景是需要验证输入，以确保提交的数据正确。

在本活动中，你将学习如何使用响应式表单验证表单。

  1. 导入 Validators

    Angular 提供了一组验证工具。要使用它们，首先更新组件以从 @angular/forms 导入 Validators。

          
    import {ReactiveFormsModule, Validators} from '@angular/forms';@Component({...})export class AppComponent {}

  2. 为表单添加验证

    每个 FormControl 都可以传入你想要用于验证 FormControl 值的 Validators。例如，如果你想让 profileForm 中的 name 字段成为必要字段，则可以使用 Validators.required。 对于 Angular 表单中的 email 字段，我们希望确保它不为空，并且遵循有效的电子邮件地址结构。我们可以通过在一个数组中组合 Validators.required 和 Validators.email 验证器来实现这一点。 更新 name 和 email FormControl：

          
    profileForm = new FormGroup({  name: new FormControl('', Validators.required),  email: new FormControl('', [Validators.required, Validators.email]),});

  3. 在模板中检查表单验证

    要确定表单是否有效，FormGroup 类具有 valid 属性。 你可以使用此属性来动态绑定 Attribute。更新提交 button，使其根据表单的有效性启用。

          
    <button type="submit" [disabled]="!profileForm.valid">Submit</button>

现在你已经了解了关于验证如何与响应式表单一起工作的基础知识。

出色地学习了在 Angular 中使用表单的这些核心概念。 如果你想了解更多信息，请务必参考 Angular 表单文档。
