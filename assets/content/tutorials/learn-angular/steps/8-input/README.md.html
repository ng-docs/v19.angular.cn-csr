使用 @Input 的组件通信

edit

有时应用开发需要你将数据发送到组件中。这些数据可以用来定制组件，或者从父组件向子组件发送信息。

Angular 使用一个叫做 Input 的概念。这类似于其他框架中的 props。要创建一个 Input 属性（Property），请使用 @Input 装饰器。

在本活动中，你将学习如何使用 @Input 装饰器向组件发送信息。

要创建一个 Input 属性（Property），请将 @Input 装饰器添加到组件类的属性（property）上：

user.component.ts

      
class UserComponent {  @Input() occupation = '';}

当你准备好通过 Input 传入一个值时，可以使用 Attribute 语法在模板中设置值。这是一个范例：

app.component.ts

      
@Component({  ...  template: `<app-user occupation="Angular Developer"></app-user>`})class AppComponent {}

请确保在你的 UserComponent 中绑定属性（property） occupation。

user.component.ts

      
@Component({  ...  template: `<p>The user's occupation is {{occupation}}</p>`})

  1. 定义一个 @Input 属性（Property）

    更新 user.component.ts 中的代码，在 UserComponent 上定义一个名为 name 的 Input 属性（Property）。现在，将初始值设置为 empty string。请务必更新模板以在句子末尾插值 name 属性（property）。

  2. 向 @Input 属性（Property）传递一个值

    更新 app.component.ts 中的代码，传入 name 属性（property），其值为 "Simran"。

    当代码成功更新后，应用将显示 The user's name is Simran。

虽然这很棒，但这只是组件通信的一个方向。如果你想从子组件向父组件发送信息和数据呢？查看下一课以了解更多信息。
