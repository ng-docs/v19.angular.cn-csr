定义路由

edit

既然你已经建立好应用来使用 Angular Router，现在你需要定义路由。

在本活动中，你将学习如何在你的应用中添加和配置路由。

  1. 在 app.routes.ts 中定义路由

    在你的应用中，有两个页面要显示：（1）首页和（2）用户页面。

    要定义路由，请在 app.routes.ts 的 routes 数组中添加一个路由对象，其中包含：

    • 路由的 path（它会自动从根路径（即 /）开始）
    • 你希望路由显示的 component
          
    import {Routes} from '@angular/router';import {HomeComponent} from './home/home.component';export const routes: Routes = [  {    path: '',    component: HomeComponent,  },];

    上面的代码是如何将 HomeComponent 添加为路由的示例。现在继续在 playground 中实现它以及 UserComponent。

    为 UserComponent 的路径使用 'user'。

  2. 向路由定义添加标题

    除了正确定义路由之外，Angular Router 还使你能够通过将 title 属性添加到每个路由，在用户导航时设置页面标题。

    在 app.routes.ts 中，将 title 属性添加到默认路由 (path: '') 和 user 路由。这是一个范例：

          
    import {Routes} from '@angular/router';import {HomeComponent} from './home/home.component';export const routes: Routes = [  {    path: '',    title: 'App Home Page',    component: HomeComponent,  },];
