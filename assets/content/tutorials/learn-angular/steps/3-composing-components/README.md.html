组合组件

edit

你已经学会了更新组件模板、组件逻辑和组件样式，但是如何在你的应用中使用组件呢？

组件配置的 selector 属性为你提供了一个名称，以便在另一个模板中引用该组件。你可以像使用 HTML 标签一样使用 selector，例如 app-user 在模板中会是 <app-user />。

在本活动中，你将学习如何组合组件。

在这个范例中，有两个组件 UserComponent 和 AppComponent。

  1. 添加对 UserComponent 的引用

    更新 AppComponent 模板，以包含对使用选择器 app-user 的 UserComponent 的引用。请务必将 UserComponent 添加到 AppComponent 的 imports 数组中，这将使其可在 AppComponent 模板中使用。

          
    template: `<app-user />`,imports: [UserComponent]

    现在，该组件显示消息 Username: youngTech。你可以更新模板代码以包含更多标记。

  2. 添加更多标记

    因为你可以在模板中使用任何你想要的 HTML 标记，所以尝试更新 AppComponent 的模板，使其也包含更多 HTML 元素。此范例将添加一个 <section> 元素作为 <app-user> 元素的父元素。

          
    template: `<section><app-user /></section>`,

你可以使用任意数量的 HTML 标记和组件，以将你的应用想法变为现实。你甚至可以在同一页面上拥有组件的多个副本。
