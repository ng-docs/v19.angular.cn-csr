使用 *ngFor* 在组件中列出对象

edit

本教程课程演示了如何在 Angular 模板中使用 ngFor 指令，以便在模板中显示动态重复的数据。

你将学到什么

  • 你将向此应用添加一个数据集
  • 你的应用将使用 ngFor 显示来自新数据集的元素列表

ngFor 的概念预览

在 Angular 中，ngFor 是一种特定类型的指令，用于在模板中动态重复数据。在纯 JavaScript 中，你会使用 for 循环 - ngFor 为 Angular 模板提供了类似的功能。

你可以利用 ngFor 迭代数组，甚至异步值。在本课程中，你将添加一个新的数据数组以进行迭代。

有关更深入的解释，请参阅内置指令指南。
