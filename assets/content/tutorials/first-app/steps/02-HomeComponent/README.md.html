创建 Home 组件

edit

本教程课程演示了如何为你的 Angular 应用创建一个新的组件。

你将学到什么

你的应用有了一个新组件：HomeComponent。

Angular 组件的概念预览

Angular 应用是围绕组件构建的，组件是 Angular 的构建块。 组件包含代码、HTML 布局和 CSS 样式信息，这些信息提供了应用中元素的功能和外观。 在 Angular 中，组件可以包含其他组件。应用的各项功能和外观可以划分并分割成组件。

在 Angular 中，组件具有元数据来定义其属性（Property）。 当你创建 HomeComponent 时，你将使用以下属性（Property）：

  • selector：用于描述 Angular 如何在模板中引用组件。
  • standalone：用于描述组件是否需要 NgModule。
  • imports：用于描述组件的依赖项。
  • template：用于描述组件的 HTML 标记和布局。
  • styleUrls：用于在数组中列出组件使用的 CSS 文件的 URL。

  1. 创建 HomeComponent

    在此步骤中，你将为你的应用创建一个新组件。

    在 IDE 的 终端 窗格中：

    1. 在你的项目目录中，导航到 first-app 目录。

    2. 运行此命令以创建一个新的 HomeComponent

            
      ng generate component home

    3. 运行此命令以构建并为你的应用启动服务器。

      注意：此步骤仅适用于你的本地环境！

            
      ng serve

    4. 打开浏览器并导航到 http://localhost:4200 以查找该应用。

    5. 确认应用构建时没有错误。

      提示：它应该渲染得和上一课中一样，因为即使你添加了一个新组件，你还没有将其包含在应用的任何模板中。

    6. 在完成后续步骤时，保持 ng serve 运行。

  2. 将新组件添加到应用的布局中

    在此步骤中，你将新组件 HomeComponent 添加到应用的根组件 AppComponent 中，以便它显示在应用的布局中。

    在 IDE 的 编辑 窗格中：

    1. 在编辑器中打开 app.component.ts。

    2. 在 app.component.ts 中，通过将此行添加到文件级导入来导入 HomeComponent。

      在 src/app/app.component.ts 中导入 HomeComponent

            
      import {Component} from '@angular/core';import {HomeComponent} from './home/home.component';@Component({  selector: 'app-root',  imports: [HomeComponent],  template: `    <main>      <header class="brand-name">        <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />      </header>      <section class="content">        <app-home></app-home>      </section>    </main>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

    3. 在 app.component.ts 的 @Component 中，更新 imports 数组属性并添加 HomeComponent。

      在 src/app/app.component.ts 中替换

            
      import {Component} from '@angular/core';import {HomeComponent} from './home/home.component';@Component({  selector: 'app-root',  imports: [HomeComponent],  template: `    <main>      <header class="brand-name">        <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />      </header>      <section class="content">        <app-home></app-home>      </section>    </main>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

    4. 在 app.component.ts 的 @Component 中，更新 template 属性以包含以下 HTML 代码。

      在 src/app/app.component.ts 中替换

            
      import {Component} from '@angular/core';import {HomeComponent} from './home/home.component';@Component({  selector: 'app-root',  imports: [HomeComponent],  template: `    <main>      <header class="brand-name">        <img class="brand-logo" src="/assets/logo.svg" alt="logo" aria-hidden="true" />      </header>      <section class="content">        <app-home></app-home>      </section>    </main>  `,  styleUrls: ['./app.component.css'],})export class AppComponent {  title = 'homes';}

    5. 将更改保存到 app.component.ts。

    6. 如果 ng serve 正在运行，应用应该会更新。 如果 ng serve 没有运行，请重新启动它。 应用中的 Hello world 应该会从 HomeComponent 更改为 home works!。

    7. 在浏览器中检查正在运行的应用，并确认应用已更新。

  3. 添加特性到 HomeComponent

    在此步骤中，你将为 HomeComponent 添加特性。

    在上一步中，你将默认的 HomeComponent 添加到应用的模板中，因此其默认 HTML 出现在应用中。 在此步骤中，你将添加一个搜索过滤器和按钮，它们将在后面的课程中使用。 目前，这就是 HomeComponent 的全部内容。 请注意，此步骤仅将搜索元素添加到布局中，但尚不具备任何功能。

    在 IDE 的 编辑 窗格中：

    1. 在 first-app 目录中，在编辑器中打开 home.component.ts。

    2. 在 home.component.ts 的 @Component 中，使用此代码更新 template 属性。

      在 src/app/home/home.component.ts 中替换

            
      import {Component} from '@angular/core';import {CommonModule} from '@angular/common';@Component({  selector: 'app-home',  imports: [CommonModule],  template: `    <section>      <form>        <input type="text" placeholder="Filter by city" />        <button class="primary" type="button">Search</button>      </form>    </section>  `,  styleUrls: ['./home.component.css'],})export class HomeComponent {}

    3. 接下来，在编辑器中打开 home.component.css 并使用这些样式更新内容。

      注意：在浏览器中，这些可以放在 src/app/home/home.component.ts 的 styles 数组中。

      在 src/app/home/home.component.css 中替换

            
      .results {  display: grid;  column-gap: 14px;  row-gap: 14px;  grid-template-columns: repeat(auto-fill, minmax(400px, 400px));  margin-top: 50px;  justify-content: space-around;}input[type="text"] {  border: solid 1px var(--primary-color);  padding: 10px;  border-radius: 8px;  margin-right: 4px;  display: inline-block;  width: 30%;}button {  padding: 10px;  border: solid 1px var(--primary-color);  background: var(--primary-color);  color: white;  border-radius: 8px;}@media (min-width: 500px) and (max-width: 768px) {  .results {      grid-template-columns: repeat(2, 1fr);  }  input[type="text"] {      width: 70%;  }   }@media (max-width: 499px) {  .results {      grid-template-columns: 1fr;  }    }

    4. 确认应用构建时没有错误。 你应该在你的应用中找到过滤器查询框和按钮，并且它们应该已设置样式。 在继续下一步之前，请纠正任何错误。

摘要： 在本课程中，你为你的应用创建了一个新组件，并为其添加了一个过滤器编辑控件和按钮。
